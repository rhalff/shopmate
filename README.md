# Shopmate App 

Proof of concept application against the turing.com api.

## Setup 

- flutter bloc
- validations
- jaguar serializer
- jaguar orm
- retro fit

## Install

```bash
flutter pub get
flutter pub run

```

### Build

`flutter packages pub run build_runner build`
